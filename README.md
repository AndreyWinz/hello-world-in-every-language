## üöÄ How it works

Each file in this repo prints:

Hello, World!

(or the closest equivalent in that language)

File names follow this format:

hello.(extension)

Examples:

- `hello.py`
- `hello.js`
- `hello.cpp`
- `hello.r`
- `hello.bf`
- `hello.whatever`
